# 🛡️ FraudGuard AI

FraudGuard AI is a real-time credit card fraud detection system built using Machine Learning. It leverages **XGBoost** and **Isolation Forest** to detect fraudulent transactions with high accuracy and provides an interactive dashboard for simulation and analysis.

---

## 🚀 Features

* 🔍 Real-time transaction fraud detection
* 📊 Confidence score & anomaly insights
* ⚡ Powered by XGBoost & Isolation Forest
* 🎛️ Interactive dashboard for simulation
* 🧪 Generate normal & fraudulent transactions
* 🧠 AI-based decision engine

---

## 🖥️ Demo

* Live transaction stream simulation
* AI analysis with instant approval/rejection
* Adjustable transaction parameters

---

## 🏗️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **ML Models:**

  * XGBoost
  * Isolation Forest
* **Libraries:** Pandas, NumPy, Scikit-learn

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/dapavan71/FraudGuard-AI.git
cd FraudGuard-AI
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

---

## 📊 How It Works

* Transaction data is processed in real-time
* Features are analyzed using trained ML models
* Isolation Forest detects anomalies
* XGBoost predicts fraud probability
* Final decision is displayed with confidence score
